#jaylist is an simple and easy key-value list.

Annotated source code: http://benbscholz.github.com/list.html

###Usage:

####Create the list:
    var mylist = new List(); 
    
####Add a value to the list:
    mylist.add("key", value); 
    
####Get the value from the list:
    var val = mylist.get("key");
